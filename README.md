# # Bunker

Веб-адаптация социальной игры **«Бункер»**, написанная с использованием **React** и **FastAPI**.

Проект позволяет генерировать игровые карточки, управлять рероллами характеристик и в будущем поддерживать полноценные онлайн-лобби с ведущим и синхронизацией игроков.

---

# Возможности

## Реализовано

* Генерация карточек игроков
* Генерация карточек бункера
* Генерация карточек катастроф
* Система рероллов для 5 характеристик игрока
* Интерфейс для локальной игры с друзьями
* Backend API для генерации игровых данных

---

## Планируется

* Система лобби
* Создание и подключение к комнате
* Авторизация по нику
* Роли игроков и ведущего
* Управление этапами игры со стороны ведущего
* Общие карточки бункера и катастрофы внутри лобби
* Синхронизация игроков в реальном времени
* Поддержка WebSocket
* Полноценные игровые сессии

# Запуск проекта

## Frontend

```bash id="frontstart"
cd frontend
npm install
npm run dev
```

## Backend

```bash id="backstart"
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

# Как это работает

Приложение генерирует основные карточки, необходимые для игры в «Бункер»:

* карточки игроков
* карточки бункера
* карточки катастроф

Игрок может выполнять реролл выбранных характеристик, а будущая система лобби позволит ведущему управлять этапами игры и синхронизировать игровой процесс между всеми участниками.
